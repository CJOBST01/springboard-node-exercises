# Springboard Node Exercises

Solutions for the two Node.js coding exercises in Unit 19.1: Introduction to Node.js.

## Layout

- `node-intro/` — A small `cat`/`webCat` CLI that reads files or URLs and optionally writes the output to a file.
- `node-markov/` — A Markov chain text generator with a `makeText.js` driver script.

Each folder is its own self-contained npm project. `node_modules` is gitignored.
